# 🌱 LifeGreen

## 📌 Sobre o Projeto

LifeGreen é uma plataforma de e-commerce de produtos orgânicos desenvolvida para conectar pequenos produtores rurais a consumidores interessados em alimentação saudável e sustentável.

A proposta do sistema é permitir o cadastro de produtores, gerenciamento de produtos, realização de pedidos, assinaturas de planos e acompanhamento de compras, proporcionando uma experiência simples, acessível e segura para todos os usuários.

Este projeto está sendo desenvolvido para fins acadêmicos como parte das atividades da disciplina.

## 🎯 Objetivos do Projeto

* Incentivar o consumo de produtos orgânicos;
* Aproximar produtores locais dos consumidores;
* Facilitar o processo de compra e venda de produtos agrícolas;
* Aplicar conceitos de desenvolvimento de software, modelagem de dados e arquitetura de sistemas.

---

## 🏗️ Estado Atual do Projeto

Atualmente o projeto encontra-se em fase de planejamento e documentação arquitetural.

Nesta etapa foram desenvolvidos:

* Diagramas C4 (Contexto, Containers e Componentes);
* Modelagem de Dados (DER e Modelo Relacional);
* Estrutura do banco de dados em SQL;
* Documentação técnica do sistema;
* Planejamento da arquitetura do back-end;
* Definição dos requisitos funcionais e não funcionais.

Algumas funcionalidades descritas nos diagramas e documentações ainda não foram implementadas de forma completa, servindo neste momento como proposta arquitetural para futuras etapas do desenvolvimento.

Dessa forma, parte dos componentes do back-end, integrações externas e funcionalidades avançadas encontram-se documentadas conceitualmente, mas ainda não estão totalmente integradas à aplicação.
