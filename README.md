# 🌱 LifeGreen

## 📌 Descrição

LifeGreen é um e-commerce de produtos orgânicos que conecta pequenos produtores a consumidores. A plataforma permite cadastro, venda de produtos, carrinho de compras, pedidos e entregas, promovendo uma experiência simples e acessível. Projeto desenvolvido para fins acadêmicos.

## 🚀 Instruções de uso

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/lifegreen-ecommerce.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd lifegreen-ecommerce
   ```

3. Abra o arquivo `index.html` no navegador (caso seja front-end simples)

4. Caso utilize backend futuramente:

   * Instale as dependências
   * Execute o servidor conforme a tecnologia utilizada

## 📁 Estrutura (exemplo)

* /src → código-fonte
* /assets → imagens e arquivos estáticos
* index.html → página inicial
